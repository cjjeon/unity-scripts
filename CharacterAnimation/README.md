This example will not help on creating animation and only used when you already have animator in prefab.

1. Add Animator to Character
2. You can add animator tab by going Window -> Animation -> Animator
3. In Animator tab, you will see Layer and Parameter Tab
4. Layer Tab is for showing animation patterns
5. Parameter is something that would trigger which state the Animation should be at
6. In Layer Tab, if you click on the connection line, you will see the condition (parameter values) when animation state changes.
